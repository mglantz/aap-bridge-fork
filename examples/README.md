# Examples

This directory contains example configurations and playbooks for AAP Bridge.

## AAP Config-as-Code

The `aap-config-as-code/` directory contains example configuration-as-code files:

- Organizations, teams, users setup
- Credentials configuration
- Projects and inventories
- Job templates and workflows
- LDAP settings
- Automation Hub configuration

See [aap-config-as-code/README.md](aap-config-as-code/README.md) for details.

## Usage

These are **examples only** - customize them for your environment before use.
